# roller-coaster-backend
This backend is paired with [react-redux-router-blog-frontend](https://github.com/rogerwschmidt/react-redux-router-blog-frontend)


## To install
```
npm install

cp .env.sample .env

createdb blog_posts

npm run knex migrate:latest

npm run knex seed:run

```

## To run
```
npm start
```
